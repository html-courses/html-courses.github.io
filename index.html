    ABOUT

    Baron Bot is a League of Legends Discord bot. It tries to offer the most accurate information, in the shortest time possible. The bot's objective is to eliminate the effort to open third party websites to get crucial information.

    TFT FEATURES

    Looking for some Teamfight Tactics features and statistics?

    COMMANDS

    COMMANDS DESCRIPTION USAGE
    !b General info and stats. !b
    !b ability Detailed information of an ability. !b ability [champion] [p/q/w/e/r/ult]
    !b add/remove Add/Remove your LoL summoner to your account. !b add [region] [summoner] / !b remove
    !b bans Most banned champions. !b bans
    !b best Best champion for specified role. !b best [role]
    !b champion Everything you need to know about a champion: Builds, runes, spells, etc. !b champion [champion] [role] [highest/frequent](optional)
    !b counters Best counters to specified champion. !b counters [champion] [role]
    !b disable/enable Disables Baron Bot on a channel. / Enables Baron Bot back on a channel. !b disable/enable
    !b help If you ever need help. !b help
    !b info Champion info. !b info [champion]
    !b invite Get the bot's invite link to add it to your server. !b invite
    !b item Detailed information of an item. !b item [item]
    !b language Check available languages and change the bot's laguange. !b language
    !b lastplayed Shows when a player last played a specified champion. !b lastplayed
    !b live Live game informations: Ranks, winrate, etc. !b live
    !b mastery List of champions with most mastery points by a summoner. !b mastery
    !b patchnotes Latest patch notes. !b patchnotes
    !b playlists Customized playlists for the chosen play style. !b playlists [category]
    !b profile Summoner profile with ranks, champions, last game, etc. !b profile
    !b rotation Free week rotation. !b rotation
    !b rune Details for a given rune. !b rune [rune]
    !b sales Current champion and skin sales. !b sales
    !b serverupdates Warnings for server updates. !b serverupdates
    !b skins Lists champions skins or displays detailed info for a certain skin along with their prices. !b skins [champion/skin name]
    !b status Server status. !b status
    !b streams Lists current livestreams for the given champion. !b streams [champion]
    !b synergy Best champions that the specified champion synergizes with. !b synergy [champion] [role]
    !b tips Tips to play with and against a champion. !b tips [champion]
    !b topplayers Players with the most mastery points all over the world on given champion. !b topplayers [champion]
    !b ultimatebravery Random summoners, build, skill path, etc. !b ultimatebravery
